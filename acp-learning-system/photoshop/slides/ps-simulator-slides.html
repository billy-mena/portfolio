Ps-3.2a–e Understand

Concept: All 5 mask types — when to use each

1
Pixel mask
Painted with Brush — black hides, white reveals, grey = partial. Best for: removing backgrounds, isolating subjects. Paint to fix.
2
Vector mask
Uses a path — resolution-independent, crisp edges at any size. Best for: logos, geometric shapes, text effects. Edit path to fix.
3
Clipping mask
Clips upper layer to shape of layer below. Best for: image inside shape, texture inside text. Alt+click border between layers.
4
Frame tool (K)
Placeholder frame — drag image in, auto-clips. Best for: layout placeholders, quick image-in-shape. Simplest clipping workflow.
5
Mask management: disable, unlink, delete
Shift+click = disable (red X). Click chain icon = unlink. Drag to trash = delete permanently.

Ps-6.2a/b Understand

Concept: File formats, bit depth & colour space

PSD
Native Photoshop format
Preserves all layers, masks, Smart Objects, adjustment layers. Always save your master as PSD. File → Save As → Photoshop.
PNG
Lossless + transparency — web standard
Supports full alpha transparency. Lossless compression. sRGB colour space for web. Use for logos, icons, UI elements.
JPG
Lossy — no transparency — photographs
Quality 80–85 = professional web standard. No transparency. Best for full-colour photographs where file size matters.
TIFF
Lossless + 16-bit + transparency — print master
Supports transparency, lossless LZW compression, and 16-bit depth. The only format supporting all three. Used for print-ready masters.
RGB
sRGB for web · Adobe RGB for print
Convert to sRGB before web publishing. Adobe RGB has wider gamut but must be converted — other profiles display with muted colours in browsers.
