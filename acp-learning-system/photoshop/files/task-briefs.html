Ps-2.3a Open and place assets as linked Smart Objects Apply ~7 min

  1. 1Use File → Place Linked (not Place Embedded) to insert the logo file.
  2. 2Select the logo file and click Place.
  3. 3Commit the transform by pressing Enter or clicking the ✓ in the Options Bar.
  4. 4Verify the linked Smart Object icon in the Layers panel, then open Window → Links.

Domain 3 — Layers, Masks & Adjustments

Ps-3.2b Create a pixel mask to remove a background Apply ~9 min
Remove the background of a product photograph non-destructively using a pixel mask. The original pixels must remain intact and the mask must remain editable at all times.
  1. 1Select the "Product" layer in the Layers panel.
  2. 2Click the Add Layer Mask button at the bottom of the Layers panel to add a white pixel mask.
  3. 3Click the white mask thumbnail to target the mask (not the layer).
  4. 4Select the Brush tool (B), set foreground colour to black (D to reset). Paint over the background to hide it.
  5. 5Use Select → Select and Mask to refine the edge around the subject.
Ps-3.3a Apply adjustment layers — Levels and Hue/Saturation Apply ~9 min
Correct an underexposed, desaturated photo using non-destructive adjustment layers. Limit both adjustments to affect only the background layer, not the entire document.
Levels — White Input
210
Hue/Sat — Saturation
+35
Clipping
Yes — background only
  1. 1Open Window → Adjustments and click the Levels icon to add a Levels adjustment layer.
  2. 2In the Properties panel, set the white input level to 210.
  3. 3Add a Hue/Saturation adjustment layer. Set Saturation to +35.
  4. 4Alt+click (Option+click) the border between both adjustment layers and the layer below to clip them.

Domain 4 — Adding Content

Ps-4.1c ✦ AI tools — Generative Fill & Sky Replacement Apply ~8 min
A landscape photo has an overcast sky and an unwanted sign in the foreground. Replace the sky and remove the sign using Photoshop AI tools. All results must be non-destructive.
  1. 1Edit → Sky Replacement. Select a Sunsets preset. Click OK.
  2. 2Select the unwanted sign using any selection tool.
  3. 3Edit → Generative Fill. Leave the text prompt EMPTY. Click Generate.
  4. 4Use the arrows in the contextual taskbar to choose the best of the 3 AI variations.

Domain 5 — Modifying Visual Elements

Ps-5.1a/b/c Selections — Subject, Color Range & save as alpha channel Apply ~9 min
Isolate a subject from a complex background using AI-based and colour-range selection methods. Refine the selection edge and save it as a reusable alpha channel.
  1. 1Select → Subject. AI detects the main subject. Marching ants appear.
  2. 2Select → Color Range. Click the sky. Set Fuzziness to 40. Click OK.
  3. 3Hold Shift and brush to add to selection. Hold Alt/Option to subtract regions.
  4. 4Select → Modify → Feather. Radius: 5 pixels.
  5. 5Select → Save Selection. Name it and click OK. Stored as an alpha channel — load via Channels panel or Select → Load Selection.
Ps-5.2a ✦ AI Remove tool, Clone Stamp & Find Distractions Apply ~9 min
Remove three types of unwanted elements from a photograph: a blemish (Spot Healing), a texture area (Clone Stamp), and background distractions (AI Remove + Find Distractions).
  1. 1Press J — Spot Healing Brush. Options Bar: Mode = Normal, Type = Content-Aware. Click on the blemish.
  2. 2Press S — Clone Stamp. Alt+click clean source area. Paint over the problem area.
  3. 3Shift+J to cycle to the Remove tool. Enable "Create New Layer" in Options Bar. Paint over the unwanted object and release.
  4. 4Edit → Find Distractions. Photoshop scans and marks all distracting elements. Click Remove Distractions.
Ps-5.3a/b Non-destructive crop, Canvas Size & Free Transform with Warp Apply ~8 min
Prepare an image for Instagram: apply a non-destructive square crop, add a white border with Canvas Size, then apply Warp to a layer element.
Crop ratio
1:1 (Square)
Delete pixels
UNCHECKED
Canvas addition
+200px W and H
Transform
Warp (right-click in Free Transform)
  1. 1Press C (Crop). Options Bar: set ratio to 1:1. UNCHECK "Delete Cropped Pixels".
  2. 2Drag to frame composition. Press Enter to commit. Hidden pixels are preserved.
  3. 3Image → Canvas Size. Add 200px to both width and height. Anchor: centre. Click OK.
  4. 4Select the logo layer. Press Ctrl+T (Free Transform). Right-click inside bounding box → Warp. Drag mesh points. Press Enter.
Ps-5.4a/b Smart Filters & layer styles — apply, copy, scale Apply ~8 min
Apply a Smart Filter to a Smart Object, add Drop Shadow and Bevel & Emboss styles, then copy and paste those styles to a second layer and scale them proportionally.
  1. 1Right-click the layer → Convert to Smart Object.
  2. 2Filter → Blur → Gaussian Blur. Radius: 3.0 pixels. Click OK. Smart Filter appears sub-layer.
  3. 3Click fx at Layers panel bottom → Drop Shadow. Enable Bevel & Emboss in same dialog. Click OK.
  4. 4Right-click the fx badge → Copy Layer Style. Click the second layer → Paste Layer Style.
  5. 5Right-click fx on second layer → Scale Effects → set to 80%.

Ps-6.1a/b Export artboards at 1x & 2x, Layers to Files, Quick Export preference Apply ~7 min
Export 3 social media artboards as PNG at standard and Retina resolutions, export individual layers as separate files, and configure the Quick Export format preference.

  1. 1File → Export → Export As. Format: PNG. Check "Use Artboards". Click Add Scale → 2x.
  2. 2Click Export All. Choose destination folder. Files are named using artboard names.
  3. 3File → Export → Layers to Files. Set file type. Click Run.
  4. 4Edit → Preferences → Export. Set Quick Export Format to JPEG. Click OK.
Ps-6.2a/b Save formats — PSD master, Save a Copy, format selection Evaluate ~7 min
Save a layered document as a PSD master, produce a flat JPEG copy without altering the working file, and identify the correct output format for three different delivery scenarios.
Master format
PSD (Save As)
Flat copy
JPEG via Save a Copy
Web transparency
PNG
Print master
TIFF 16-bit
  1. 1File → Save As → Photoshop (.psd). This preserves all layers — master file.
  2. 2File → Save a Copy → JPEG. Saves a flat copy; the PSD stays open and unchanged.
  3. 3For web graphic with transparency: use PNG (lossless, supports alpha channel).
  4. 4For print retouching master: TIFF at 16-bit (lossless, transparency, maximum tonal range).
  5. 5Before web delivery: Image → Mode → assign sRGB profile. Adobe RGB shifts colours in browsers.

Ps-6.2a/b — Save formats

Step 1
Save master
File → Save As → Photoshop (.psd)
Preserves all layers, Smart Objects, adjustment layers, masks.
Step 2
Save flat copy
File → Save a Copy → JPEG
Critical: "Save a Copy" does NOT change the active document. "Save As" JPEG would switch the working document to JPEG.
Step 3
Format for web transparency
PNG
PNG = lossless compression + full alpha transparency. JPEG has no transparency support.
Step 4
Format for print master
TIFF at 16-bit
TIFF: lossless LZW compression + transparency + 16-bit. Only format supporting all three for print.
