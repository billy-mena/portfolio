Domain 2 — Document Setup & Interface

Id-2.1 New document — A4 facing pages, bleed, slug, columns Apply ~9 min
Create a print magazine document with the following exact specifications. All settings must be applied in the New Document dialog before clicking Create.
Intent
Print
Size
A4 (210 × 297mm)
Pages / Facing
4 pages, Facing Pages ✓
Margins
12mm all sides
Columns / Gutter
2 columns, 5mm gutter
Bleed
3mm all sides
Slug Top
6mm
  1. 1File → New → Document (Ctrl+N). Enable the Preview checkbox.
  2. 2Intent: Print. Width: 210mm, Height: 297mm. Pages: 4. Check Facing Pages.
  3. 3Margins: click the chain icon to link all four fields. Set all to 12mm.
  4. 4Columns: 2. Gutter: 5mm.
  5. 5Bleed: click More Options if needed. Set all bleed fields to 3mm (chain icon to link). Slug Top: 6mm.
  6. 6Click Create. Verify: red guides = bleed, purple guides = margins, green = slug.
Id-2.3 Place assets & manage the Links panel Apply ~8 min
Place two image files, verify their link status in the Links panel, relink a missing file, and use Edit Original to open a placed Photoshop file.
  1. 1File → Place (Ctrl+D). Select a photograph → Open. Click on the page to place at image size.
  2. 2Window → Links. Verify the chain icon (🔗) next to the placed file = linked and current.
  3. 3Select the missing link (red ?) in Links panel. Click the Relink button (chain + arrow icon) → navigate to file → Open.
  4. 4Select any linked image → click the Edit Original button (pencil icon) in the Links panel.

Domain 3 — Organising Documents

Id-3.1 Create & apply a Master page — override on one page Apply ~9 min
Create an "A-Running Footer" master page containing a footer text frame with an automatic page number. Apply it to all 12 pages. Override the footer on page 3 for a custom message.
  1. 1Window → Pages (F12). Panel menu ≡ → New Master. Name: Running Footer, Prefix: A → OK.
  2. 2Double-click A-Running Footer to enter it. Press T, drag a footer text frame at the bottom.
  3. 3Click inside the frame. Type → Insert Special Character → Markers → Current Page Number.
  4. 4Panel ≡ → Apply Master to Pages → A-Running Footer → Pages: 1–12 → OK.
  5. 5Navigate to page 3. Ctrl+Shift+click the greyed footer frame to override it. Edit the text.
Id-3.4 Object styles — create, apply, redefine Apply ~7 min
Create an Object Style "Caption Frame" from a manually formatted text frame (1pt stroke, 4mm inset). Apply to three frames, then redefine the style by updating the stroke to 2pt.
  1. 1Select a text frame. Set: Fill: None, Stroke: 1pt Black. Object → Text Frame Options → Inset: 4mm all → OK.
  2. 2Window → Styles → Object Styles (Ctrl+F7). With the frame selected, click + → name: Caption Frame → OK.
  3. 3Select three other text frames → click Caption Frame in the Object Styles panel.
  4. 4Edit one frame locally (stroke: 2pt). Panel ≡ → Redefine Style → all instances update.

Domain 4 — Text & Tables

Id-4.1 Thread text frames & Text Frame Options Apply ~8 min
Resolve a text overflow on page 1 by threading to a new frame on page 2. Then configure a sidebar frame with 6mm inset, 2 columns, and centred vertical alignment.
  1. 1Click the red + out-port on the overflowing frame. Navigate to page 2. Click to place the threaded frame.
  2. 2View → Extras → Show Text Threads to confirm the thread connection.
  3. 3Select the sidebar frame. Object → Text Frame Options (Ctrl+B).
  4. 4Inset Spacing: chain icon → all 6mm. Columns: 2, Gutter: 4mm. Vertical Justification: Center → OK.
Id-4.2 Paragraph Styles & Character Styles Apply ~9 min
Create Body Text and Heading 1 Paragraph Styles. Set Next Style on Heading 1 to Body Text. Create an "Emphasis" Character Style (bold + brand colour). Apply all styles throughout the article.
  1. 1Window → Styles → Paragraph Styles (F11). Format body text manually → click + → name: Body Text → OK.
  2. 2Format a heading → + → name: Heading 1 → Next Style: Body Text → OK.
  3. 3Window → Styles → Character Styles (Shift+F11). Select emphasis text (bold + colour) → + → name: Emphasis → OK.
  4. 4Apply styles: click inside paragraph → click style name in panel.
Id-4.3 Insert & format a table Apply ~8 min
Insert a 5-row, 4-column table with 1 header row. Format the header with a dark fill. Apply alternating row fills and set column stroke weight to 0.5pt.
  1. 1Click inside a text frame. Table → Insert Table → Body Rows: 5, Columns: 4, Header Rows: 1 → OK.
  2. 2Click in header row → Table → Select → Row. Apply dark fill from Swatches panel.
  3. 3Table → Table Options → Alternating Fills. Set odd row colour → OK.
  4. 4Select all → Table → Cell Options → Strokes and Fills → Weight: 0.5pt → OK.

Domain 5 — Visual Elements & AI Features

Id-5.1 Place images, frame fitting & content repositioning Apply ~8 min
Draw an image frame, place a photograph, apply Fill Frame Proportionally and Auto Fit. Then reposition the image content within the frame to improve the crop composition.
Frame tool
F (Rectangle Frame)
Fitting
Fill Frame Proportionally
Auto Fit
Checked (Control bar)
Content select
Double-click (orange handles)
  1. 1Press F → drag to draw a rectangle frame. Select it → File → Place (Ctrl+D) → choose photograph.
  2. 2Right-click → Fitting → Fill Frame Proportionally. Check Auto Fit in the Control bar.
  3. 3Double-click the image → orange handles appear (content selected). Drag to reposition within the frame.
  4. 4Click outside to deselect. Single-click the frame to verify blue handles (frame selected).
Id-5.5 ✦ Text-to-Image & Generative Expand Apply AI 2025 ~7 min
Generate a full-bleed background image using a text prompt, then use Generative Expand to extend a landscape photograph to fill a taller frame.
  1. 1Draw a large empty image frame. With it selected: Object → Generate Image (or click Generate in taskbar).
  2. 2Enter prompt: "aerial mountain range, golden hour, cinematic wide format". Click Generate.
  3. 3Click the best of the 4 variations → Place → image fills the frame.
  4. 4Select a placed photograph. Drag the top frame handle to make the frame taller than the image.
  5. 5Click Generative Expand in the contextual taskbar → choose variation → Accept.

Domain 6 — Publishing

Id-6.1 / 6.2 Preflight — fix errors, create profile, package document Apply ~8 min
Resolve three Preflight errors, create a custom "Print Ready" profile that flags images below 300 ppi, then package the document for printer delivery.
  1. 1Window → Output → Preflight. Click Missing Links error → navigate to frame → Links panel → Relink.
  2. 2Click Overset Text error → navigate to frame → click red + out-port → thread to new frame.
  3. 3Preflight panel ≡ → Define Profiles → + → name: Print Ready → Links → Min. Image Resolution: 300 ppi → OK.
  4. 4File → Package (Alt+Shift+Ctrl+P) → Report → check Copy Fonts & Copy Links → fill Instructions → Create Packaged Folder.
Id-6.3 Export PDF/X-1a for print & Interactive PDF for screen Evaluate ~8 min
Export a print-ready PDF/X-1a with crop marks and bleed. Then export a separate Interactive PDF with hyperlinks and bookmarks for the client's digital edition.
Print preset
PDF/X-1a:2001
Marks
Crop + Bleed marks
Bleed
Use Document Bleed Settings
Pages
All, as Spreads
  1. 1File → Export (Ctrl+E) → Adobe PDF (Print). Preset: [PDF/X-1a:2001].
  2. 2Marks and Bleeds tab → Crop Marks ✓ → Bleed Marks ✓ → Use Document Bleed Settings ✓.
  3. 3General tab → Pages: All → Spreads ✓ → click Export.
  4. 4File → Export → Adobe PDF (Interactive) → Hyperlinks: All ✓ → Bookmarks ✓ → OK.
