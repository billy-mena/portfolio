01 — The Strategic Shift

Before this intervention, the organization relied on a high-friction, high-cost training model: memos, printed materials, and costly face-to-face sessions at head office. For a national rollout across hundreds of employees, this simply didn't scale.

⛔
Before: The Problems
  • Inconsistent messaging across regions and branches
  • Travel costs and scheduling friction for every session
  • No centralized progress monitoring for 300+ employees
  • Manual gradebooks: slow, error-prone, unscalable
  • Leadership lacked data to identify gaps until it was too late
✅
After: The Transformation
  • Single source of truth: Google Classroom + Captivate hub
  • Automated notifications and real-time grade sync
  • Drill-down BI dashboard: region, area, position level
  • Managers/Supervisors certified first as local anchors
  • Self-managing — the system ran without manual intervention
02 — The Architecture

I designed and deployed a multi-layered digital learning environment from scratch — self-taught across every tool. No vendor. No IT department. Three integrated layers working together as a single performance ecosystem.

📚
Layer 1 — The Learning Hub (Google Classroom + Adobe Captivate)
Built a centralized digital hub ensuring all 336 employees — regardless of their geographic location — received identical, high-fidelity interactive training. Captivate modules handled branching scenarios, knowledge checks, and completion tracking. Google Classroom managed assignment distribution, deadlines, and cohort management across phases.
🤖
Layer 2 — The Automated Feedback Loop (Custom Google App Scripts)
Self-taught JavaScript-based App Scripts bridged the gap between Google Forms (assessments) and real-time gradebooks. The scripts triggered automated notifications — delivering immediate scores to learners and escalation reminders for incomplete tasks. The classroom effectively managed itself: no manual follow-up required.
📊
Layer 3 — The Business Intelligence Engine (PowerQuery · DAX · VBA)
Developed the Pop Club Dashboard using Power Query, DAX, and VBA macros in Excel. Stakeholders could drill down into completion metrics by region, area head, and position level — transforming training completion from a checkbox into a measurable, actionable business metric that informed strategic decisions for Phase 2.
Tech Stack Breakdown
Adobe Captivate

Interactive eLearning modules with branching scenarios, quiz logic, and SCORM-compatible completion tracking for all 336 learners.

Google Classroom + Forms

Central hub for assignment distribution, deadlines, cohort management, and assessment data collection across all national branches.

Google App Scripts (JavaScript)

Custom automation bridging Forms → Gradebook sync, triggering real-time learner notifications and escalation alerts for incomplete tasks.

Power Query + Power Pivot + DAX

Built a relational data model from raw completion data; enabled multi-dimensional analysis by region, role, and date range for leadership reporting.

VBA (Macros)

Automated repetitive reporting tasks and data cleaning pipelines — reducing manual reporting time to near zero and ensuring data integrity.

ADDIE Framework

Full model applied: Analysis (national gap audit), Design (certification pathway), Develop (Captivate + scripts), Implement (phased rollout), Evaluate (completion KPIs + Phase 2 blueprint).
