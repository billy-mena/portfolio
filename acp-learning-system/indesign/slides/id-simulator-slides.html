Print document terms — exam essentials
Bleed
Artwork extending beyond the trim edge — typically 3mm. Prevents white borders after cutting.
Id-1.3a
Slug
Area outside the bleed. Holds printer marks, job info, colour bars. Does not print on final output.
Id-1.3b
Live area / safe zone
Area inside the margins where critical content should sit. Avoid placing text near the trim edge.
Id-1.3c
Rich black
A mix of all four CMYK inks (e.g. C40 M30 Y30 K100) for deeper blacks on large areas. Not for small text.

Task: New document — A4 facing pages with bleed & slug
Create a print magazine document: A4 facing pages, 3mm bleed all sides, 6mm top slug, 12mm margins, 2 columns, 5mm gutter.
1
File → New → Document (Ctrl+N). Enable Preview checkbox immediately.
2
Intent: Print. Width: 210mm, Height: 297mm (A4). Pages: 4. Check Facing Pages.
Facing pages = left/right spread structure for bound publications.
3
Margins: all 12mm (chain icon to link). Columns: 2, Gutter: 5mm.
4
Bleed: 3mm all sides (chain icon). Slug Top: 6mm.
Id-2.1: Bleed = extends beyond trim. Slug = holds printer marks only.
5
Click Create. Red guides = bleed. Purple = margins. Blue/green = slug.
Id-2.3 Apply ~8 min
Task: Place assets & manage Links panel
Place a photograph, verify link status, relink a missing file, and use Edit Original to modify a placed asset.
1
File → Place (Ctrl+D). Select image → Open. Click or drag on page to place.
2
Window → Links. Verify: 🔗 chain = current, ⚠ yellow = modified, ? red = missing.
3
Click a missing link (?) → Relink button → navigate to file → Open.
Relink All in panel menu if many files are missing.
4
Select link → Edit Original (pencil icon) → file opens in source app → save there → link updates.
Id-2.3: Never re-place to update — always Edit Original to keep the link intact.

Id-3.1 Apply ~9 min
Task: Create & apply Master pages
Create "A-Running Footer" master with a footer text frame and page number. Apply to all 12 pages. Override the footer on page 3.
1
Window → Pages (F12). Panel menu ≡ → New Master. Name: Running Footer, Prefix: A.
2
Double-click A-Running Footer to enter it. Draw a footer text frame (T + drag).
3
Inside frame: Type → Insert Special Character → Markers → Current Page Number.
Shows "A" on master. Resolves to correct numbers on document pages.
4
Panel ≡ → Apply Master to Pages → A-Running Footer → Pages: 1-12 → OK.
5
On page 3: Ctrl+Shift+click (Cmd+Shift) the greyed footer → overrides it locally.
Id-3.1: Override = locally editable. Other pages still use master version.

Task: Thread text & apply Paragraph Styles
Resolve overset text by threading to a new frame, then apply Body Text and Heading 1 Paragraph Styles to format a feature article.
1
Click the red + out-port (bottom-right of overflowing frame) → thread cursor loads.
2
Navigate to page 2 → click or drag to create linked frame. Text flows in.
View → Extras → Show Text Threads to see connection arrows.
3
Window → Styles → Paragraph Styles (F11). Format body paragraph, then click + to save as "Body Text".
4
Create "Heading 1" with Next Style: Body Text. Press Enter after a heading → Body Text applies automatically.
Id-4.2: Next Style automates typographic flow. Format one, create, then apply to document.

Task: Image fitting & Generative Expand
Place and fit a photograph, reposition the content within the frame, then use Generative Expand to extend a portrait image to landscape format.
1
Press F (Rectangle Frame). Drag to draw frame. Select frame → File → Place (Ctrl+D) → choose photo.
2
Right-click → Fitting → Fill Frame Proportionally. Check Auto Fit in Control bar.
Fill Frame: fills fully, may crop. Fit Content: no crop, may leave gaps.
3
Double-click image → content selected (orange handles). Drag to reposition within frame. Single-click outside to return to frame selection.
4
Resize frame wider than the photo. Generative Expand button appears in taskbar → click → choose variation → Accept.
Id-5.5: Firefly fills the empty frame area with AI content that blends with the original photo.

Task: Preflight errors & Package for print
Resolve Preflight errors in a magazine document, create a custom 300 ppi profile, then package for printer delivery.
1
Window → Output → Preflight. Red dot = errors. Click each error to navigate to it.
2
Fix missing link: Links panel → Relink. Fix overset: thread to a new frame.
3
Create custom profile: Preflight panel ≡ → Define Profiles → + → name: Print Ready → Links → min. resolution: 300 ppi → OK.
Activate the profile in the panel dropdown.
4
File → Package (Alt+Shift+Ctrl+P). Review Summary → Report → check Copy Fonts & Copy Links → add Instructions → Create Packaged Folder.
Id-6.2: Package folder contains: .indd, Document Fonts/, Links/, Instructions.txt.
Id-6.3 Apply ~8 min
Task: Export press-quality PDF/X-1a
Export a print-ready PDF with crop marks, 3mm bleed, all spreads, press-quality images. Then export a separate Interactive PDF for digital delivery.
1
File → Export (Ctrl+E) → Adobe PDF (Print). Preset: [PDF/X-1a:2001].
PDF/X-1a: CMYK only, no live transparency, most printer-compatible.
2
Marks and Bleeds tab → Crop Marks ✓ → Bleed Marks ✓ → Use Document Bleed Settings ✓.
3
General tab → Pages: All → Spreads ✓ → Export.
4
For digital: File → Export → Adobe PDF (Interactive) → Hyperlinks ✓ → Bookmarks ✓ → OK.
Id-6.3: PDF Print = CMYK, crop marks, press. PDF Interactive = hyperlinks, video, no print marks.
