Instructional design foundations

ADDIE
Analyse → Design → Develop → Implement → Evaluate. Objectives stated before instruction, formative checks embedded, summative Kirkpatrick evaluation for each tool.
Bloom's Taxonomy
Every task tagged: Remember (MC recall), Understand (concept slides), Apply (Live tasks), Evaluate (independent mode + rubric). Tasks progress from guided to independent.
Gagne's 9 Events
Each task step tagged to one of Gagne's events — Gain Attention, Stimulate Recall, Present Content, Provide Guidance, Elicit Performance, Provide Feedback, Assess, Enhance Retention.
Mayer's Multimedia + Kirkpatrick
Mayer's 12 principles guide interface design — contiguity, segmenting, coherence. Kirkpatrick L1–L3 toolkits measure Reaction, Learning, and on-the-job Behavior transfer per tool.

ACP exam format — all three tools

Section 1 — Selected response

Multiple-choice covering Domain 1: design industry concepts, copyright, terminology, principles. The shared theory module covers all of it.

Section 2 — Live-in-the-App

Tasks performed inside the real Adobe application. You receive a brief and must complete actions in the actual software. All simulators prepare you for exactly this.

2025 AI objectives

Generative Fill + Sky Replacement (Ps), Generate Vectors + Generative Shape Fill (Ai), text-to-image + Generative Expand (Id). All covered in simulators.

Duration: 50 minutes per exam  ·  Delivered by: Certiport / NCS Pearson  ·  Target: ~150 hours of instruction and hands-on practice per tool
