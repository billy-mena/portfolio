  2. 2Click the Save Preset icon. Enter the preset name and confirm.
  3. 3Click Create to generate the document.
Ai-2.3a/b Place assets — embed vs link, Links panel Apply ~7 min
Place a client logo as an embedded asset and a background texture as a linked asset into an existing Illustrator document.
  1. 1File → Place. Select the logo file. UNCHECK "Link" to embed it. Click Place.
  2. 2File → Place again. Select the texture file. CHECK "Link". Click Place.
  3. 3Window → Links. Verify: logo shows no link icon (embedded); texture shows a link icon.
  4. 4Select the linked texture in the Links panel. Click the Edit Original button (pencil icon).
Ai-2.4b/c Export for Screens & Package Apply ~8 min
Export all artboards at 1x and 2x resolution. Then package the project to collect all linked files and fonts.
  1. 1File → Export → Export for Screens.
  2. 2Verify all artboards are checked. Click "+ Add Scale" → set to 2x, format PNG.
  3. 3Click Export Artboard. Choose destination folder.
  4. 4File → Package. Choose destination. Click Package.

Domain 4 — Objects, Paths & Text

Ai-4.1e ✦ Generate Vectors — Adobe Firefly AI Apply ~7 min
Use the Generate Vectors feature to create a custom vector icon. Edit the generated result using standard Illustrator path tools.
  1. 1Edit → Generate Vectors. The AI prompt panel opens.
  2. 2Enter a descriptive prompt (subject + style + colours). Click Generate.
  3. 3Select the best variation from the 4 options. Place on artboard.
  4. 4Object → Ungroup (Ctrl+Shift+G). Press A (Direct Selection) to edit anchor points.
Ai-4.4a/b Thread text frames & convert to outlines Apply ~8 min
Thread a long article across two text frames. Then convert the headline to vector outlines for custom editing.
  1. 1Click the red + overflow indicator on the first text frame.
  2. 2Click the second text frame to flow the overflow text into it.
  3. 3Type → Area Type Options. Set Columns to 2.
  4. 4Select the headline text. Type → Create Outlines (Shift+Ctrl+O).
  5. 5Ungroup (Ctrl+Shift+G). Press A to edit individual letterform paths.

Domain 5 — Appearance & Position

Ai-5.3a/b Clipping masks & opacity masks Apply ~8 min
Create a clipping mask to constrain a pattern to a star shape. Then apply an opacity mask to a photograph for a soft fade-to-transparent edge.
  1. 1Place the star shape ON TOP of the pattern. Select both. Object → Clipping Mask → Make (Ctrl+7).
  2. 2Confirm: star path loses fill/stroke; pattern shows only within star boundary.
  3. 3Select photo + gradient shape. Window → Transparency. Panel menu (≡) → Make Opacity Mask.
  4. 4Click the mask thumbnail in the Transparency panel to activate mask editing.
