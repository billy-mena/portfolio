Task: New document settings & presets

Create a social media document: 1080×1080 px, 3 artboards, RGB, 72 ppi raster effects. Save as a preset named "Social Square 1080".
1
File → New (Ctrl+N). The New Document dialog opens with preset tabs.
2
Set Width and Height to 1080 px. Set Artboards to 3.
Units: Pixels. 1080 = Instagram square standard.
3
Colour Mode: RGB. Raster Effects: 72 ppi (Screen).
Ai-2.1a · 72 ppi = screen. 300 ppi = print. Raster Effects controls shadow/glow resolution.
4
Click the Save Preset icon. Name it "Social Square 1080" → OK → Create.

Task: Manage multiple artboards

Duplicate Artboard 1, resize Artboard 2 to 1200×630 px (Facebook), rename it, and rearrange artboards in the panel.
1
Press Shift+O — Artboard tool. Click an artboard to select it.
2
Alt+drag the artboard to duplicate it (copies artwork too).
3
With Artboard 2 selected, enter W: 1200, H: 630 in the Options Bar.
4
Window → Artboards. Double-click the name to rename it "Facebook". Drag the row to reorder.
Ai-2.2a · Artboard names become filenames when exporting — always name them first.

Ai-3.3a/bUnderstand

Concept: Global, process, and spot swatches

G
Global process colour
White triangle in corner. Edit once → all objects update. Best for brand colours.
S
Spot colour (Pantone)
Dot in corner. Pre-mixed ink — printed with a dedicated press plate. More expensive but exact colour match. Always use Pantone names.
P
Process colour (non-global)
No triangle, no dot. CMYK mix. Editing it only changes the swatch — not objects using it. Good for one-off colours.
ASE
Export colour groups as .ase
Window → Swatches → panel menu → Save Swatch Library as ASE. Share palettes with other Adobe apps.
Ai-3.5Apply~8 min

Task: Layer management

Create sublayers, reorder layers, merge two layers, collect selected objects into a new layer, and use Paste Remembers Layers.
1
Window → Layers (F7). Select Layer 1. Alt+click New Layer button → creates a sublayer.
2
Drag layers in the panel to reorder. Top = drawn in front.
3
Ctrl+click two layers → panel menu (≡) → Merge Selected.
4
Select objects → panel menu → Collect in New Layer.
5
Layers panel ≡ → Paste Remembers Layers (toggle on). Pasted objects return to their original layer.
Ai-3.5c · Without this, paste always goes to the active layer — breaks layer organisation.

Task: Pen tool & Curvature Pen

Trace a logo silhouette using the Pen tool, creating corner and curved anchor points. Convert a smooth point to a corner, then redraw a path with the Curvature Pen.
1
Press P — Pen tool. Click = corner point. Click+drag = smooth curve point.
2
Alt+click a smooth anchor point to convert it to a corner (breaks direction handles).
The Convert Anchor Point tool (Shift+C) also does conversions.
3
Close the path by clicking back on the first anchor point (cursor shows °).
4
Shift+~ — Curvature Pen. Just click: smooth curves created automatically without dragging.
Ai-4.1b · Double-click in Curvature Pen = corner point. Faster for organic, flowing shapes.

Task: Generate Vectors — Adobe Firefly

Use Generate Vectors to create a custom vector icon from a text prompt. Edit the generated vector paths with standard Illustrator tools.
1
Edit → Generate Vectors. The AI prompt panel opens.
2
Type a descriptive prompt (subject + style + colours) → click Generate.
Example: "simple mountain landscape line art, minimal style"
3
4 variations appear. Click the best one to place it on the artboard.
4
Object → Ungroup (Ctrl+Shift+G). Press A (Direct Selection) to edit individual anchor points.
Ai-4.1e · Generated vectors are actual scalable SVG paths — not raster. Fully editable with all vector tools.

Task: Pathfinder & Shape Builder

Combine and subtract shapes to create a logo mark. Use both the Pathfinder panel and the Shape Builder tool.
1
Window → Pathfinder (Shift+Ctrl+F9). Top row = Shape Modes.
2
Select both circles → Unite (merges into one shape, takes bottom object's colour).
3
Select rectangle + triangle → Minus Front (subtracts top shape from all below).
4
Shift+M — Shape Builder. Drag across regions to merge. Alt+drag to delete a region.
Ai-4.2a · Shape Builder is more interactive than Pathfinder for complex multi-shape work.
5
Select a complex path → Object → Path → Simplify. Reduce anchor points without losing shape quality.

Task: Clipping masks & opacity masks

Constrain a background pattern to a logo shape using a clipping mask. Apply an opacity mask to a photo for a soft fade edge.
1
Place the clip shape on TOP of the content. Select both. Object → Clipping Mask → Make (Ctrl+7).
2
The clipping path loses its fill/stroke. Content shows only within its boundary.
To release: Object → Clipping Mask → Release (Alt+Ctrl+7).
3
For opacity mask: select photo + gradient shape. Window → Transparency. Panel menu (≡) → Make Opacity Mask.
4
Click the mask thumbnail (right thumbnail) in the Transparency panel to edit the mask independently.
Ai-5.3b · Opacity mask: white = fully opaque, black = transparent, grey = semi-transparent.

Task: Align & distribute objects

Distribute 5 icons with equal spacing, align them to the artboard right edge, then use Key Object alignment for one fixed reference.
1
Window → Align (Shift+F7). Always check "Align to:" first.
2
Set "Align to" → Artboard. Select all. Click Distribute Spacing Horizontal (equal gaps).
Distribute Spacing = equal gaps. Distribute Objects = equal centres. Use Spacing for even gaps.
3
Click Align Right Edge → all objects move to the right artboard edge.
4
For Key Object: select all → click one object again (blue border appears = key object) → align. It stays fixed; others move to it.
Ai-5.4b · Key Object prevents that specific object from moving during alignment operations.
